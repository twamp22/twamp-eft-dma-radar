## What is this?
- This is an up-to-date build of Lone DMA EFT/Arena Radar. This is a standalone copy of the software with *No Restrictions*. There have been changes made based on Twamp's preferences.

## What is different about this fork?
- This fork has some minor Quality of Life changes for my personal workflow. Primary differences are:
  - The ESP window is no longer set to TopMost
    - TopMost caused issues when viewing other windows which need focus.
  - The ESP status text is removed
    - The checkbox was broken and it is not a needed addition to the UI.
  - The main window title no longer contains FPS
    - When using OBS, the FPS counter would cause issues capturing the ESP window.
